PayProp Disabled

PayProp integration is currently disabled.

To enable PayProp features, set PAYPROP_ENABLED=true in your environment variables and restart the application.

What is PayProp?

PayProp is a property management integration that allows you to:

  • Import property data
  • Sync property owners and tenants
  • Manage financial data
  • Generate statements
How to Enable PayProp
  1. Set environment variable: PAYPROP_ENABLED=true
  2. Configure PayProp API credentials
  3. Restart the application
  4. Visit this page again to access PayProp features
